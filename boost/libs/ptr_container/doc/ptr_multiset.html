Pointer Container Library

Class ptr_multiset

A ptr_multiset<T> is a pointer container that uses an underlying std::multiset<void*> to store the pointers.

See also:

  • reversible_ptr_container
  • associative_ptr_container
  • ptr_multiset_adapter
