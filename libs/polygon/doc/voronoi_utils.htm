Voronoi Utils

Voronoi utilities implements a set of tools that users may find useful especially for the visualization of the Voronoi diagrams or discretization of the parabolic edges.

Important

There are no warranties about precision of the output structures produced by the utilities class. This is mostly because input coordinate type the utilities operate with is supposed to be double. Any degeneracies may be reported, however those are of low priority to be fixed.

Declaration

template <typename T, typename TRAITS = voronoi_utils_traits<T> >
class voronoi_utils;

T - output coordinate type used by the utilities.
TRAITS - output geometries type traits (see definition below).

Member Functions

template <typename CT>
static brect_type scale_bounding_rectangle(
  const bounding_rectangle<CT> &brect,
  fpt_type factor = 1.0)
 Returns the scaled bounding rectangle.
The center of the transformation corresponds to the center of the input rectangle.
template <typename CT>
static void discretize_finite_edge(
  const voronoi_edge<CT> &edge,
  coordinate_type max_error,
  point_set_type &discretization)
 Provides the point discretization of the input voronoi edge.
If the edge is infinite (ray, line) returns empty output set.
Max error specifies the maximum distance between approximation and original parabolic arc.
template <typename CT>
static void clip_linear_edge(
  const voronoi_edge<CT> &edge,
  const brect_type &brect,
  point_set_type &clipped_edge)
 Clips the input Voronoi edges with the specified rectangle.
If the edge is a parabolic arc returns empty output set.
template <typename PointType>
static void clip_linear_edge(
  const PointType &p1,
  const PointType &p2,
  const brect_type &brect,
  point_set_type &clipped_edge)
 Clips the input linear edge with the specified rectangle.
The edge is defined by its endpoints.

Bounding Rectangle

The module provides implementation of a simple bounding rectangle data structure.

Declaration

template <typename T>
class bounding_rectangle;

T - coordinate type the rectangle operates with.

Member Functions

bounding_rectangle()
 Default constructor. Initializes the empty bounding rectangle.
bounding_rectangle(
    coordinate_type x1,
    coordinate_type y1,
    coordinate_type x2,
    coordinate_type y2)
 Constructs the bounding rectangle with the given coordinates.
void update(
    coordinate_type x,
    coordinate_type y)
 Extends the bounding rectangle with the specified point defined by its coordinates.
If the rectangle is not initialized, initializes it with the specifed point.
bool is_empty() const
 Returns true if the rectangle is empty (uninitialized), else false.
void clear()
 Sets rectangle to an empty state.
bool contains(
   coordinate_type x,
   coordinate_type y) const
 Returns true if the rectangle contains the given point defined by its coordinates, else false.
coordinate_type x_min() const
 Returns the x coordinate of the bottom left corner of the rectangle.
coordinate_type y_min() const
 Returns the y coordinate of the bottom left corner of the rectangle.
coordinate_type x_max() const
 Returns the x coordinate of the top right corner of the rectangle.
coordinate_type y_max() const
 Returns the y coordinate of the top right corner of the rectangle.
